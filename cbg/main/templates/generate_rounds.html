What each button does:
  • Generate Handicaps & Rounds:
    • Calculates and saves handicaps for all golfers in the season
    • Generates golfer matchups for the selected week
    • Generates Round objects for all golfer matchups
    • Calculates and stores Points for each hole
    • Updates Round totals and statistics
  • Generate Handicaps Only:
    • Calculates and saves handicaps for all golfers in the current season
    • Does NOT generate matchups or rounds
    • Useful when you want to update handicaps after new scores are entered
  • Generate Golfer Matchups Only:
    • Creates golfer matchups for the selected week only
    • Handles substitutions and A/B golfer assignments
    • Does NOT generate rounds or calculate handicaps
    • Useful when you want to set up matchups before entering scores
  • Generate Rounds Only:
    • Generates Round objects for existing golfer matchups in the selected week
    • Calculates and stores Points for each hole
    • Updates Round totals and statistics
    • Requires handicaps and matchups to already exist
    • Useful when you want to generate rounds after entering scores
  • Recalculate All Played Weeks:
    • Recalculates all fully played weeks in the current season
    • Useful for updating data after rule changes or corrections
Note: Make sure the schedule has been entered for the selected week before generating matchups or rounds.
